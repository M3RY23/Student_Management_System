
# Öğrenci Yönetim Sistemi (Student Management System)

## Proje Amacı

Bir okulun öğrenci, öğretmen ve yönetici işlemlerini yönetebilen konsol tabanlı bir uygulama geliştirilmesi.

Sistemde üç farklı kullanıcı rolü bulunacaktır:

* Admin
* Öğretmen
* Öğrenci

Her rolün farklı yetkileri olacaktır.

---

# GPA Hesaplama

Not Aralıkları:

| Not    | Katsayı |
| ------ | ------- |
| 90-100 | 4.0     |
| 80-89  | 3.0     |
| 70-79  | 2.0     |
| 60-69  | 1.0     |
| 0-59   | 0.0     |

Örnek:

```java
public double calculateGPA() {

    double total = 0;

    for(Integer grade : grades.values()) {

        if(grade >= 90)
            total += 4.0;
        else if(grade >= 80)
            total += 3.0;
        else if(grade >= 70)
            total += 2.0;
        else if(grade >= 60)
            total += 1.0;
    }

    return total / grades.size();
}
```

# Bonus Özellikler

## 1. Stream API

En başarılı öğrenciyi bulma

```java
students.stream()
        .max(Comparator.comparing(Student::calculateGPA))
        .orElseThrow();
```

---

## 2. Sıralama

Öğrencileri GPA'ya göre sırala.

```java
students.sort(
        Comparator.comparing(Student::calculateGPA)
                  .reversed()
);
```
